# Amanda(아무나 만나지 않는다)
실제 백앤드 팀과의 협업.


## About Amanda
- 첫 백앤드 팀과으 협업.
- 실제 백앤드가 제공하는 DATA를 RESTAPI 방식을 사용하여 상호간의 통신.
- 쇼셜로그인(카카오로그인, Apple로그인, 이메일로그인)울 통한 로그인.


## Role
- 백엔드팀이 제공해주는 UserData를 RESRAPI방식으로 가졍와서 프로필뷰를 주도적으로 구현.
- 프로필뷰의 UI를 담당.

- Amanda를 통해 다음과 같은 내용을 스터디할 수 있었습니다.
  1. Codable ,JSNON
  2. REST API문서 이해
  3. 쇼셜로그인(카카오로그인, Apple로그인, 이메일로그인)
  4. URLSession, dataTask
  5. 처음으로 백 앤드와의 협업을 통해 실제 회사에서 어떤 식으로 서 버팀과 협업을 하게 되는지 간접적으로 경험해 볼 수 있었습니다. 처음 백엔드 팀과 회의를 진행할 때는 어디서부터 어떤 식을 대화를 해야 되는지 막막했지만 점차 서로에 업무에 이해하기 시작하면서 목적 달성을 할 수 있었습니다. 협업을 하면서 발생하는 문제도 많았지만 그때마다 느꼈던 것은 의사소통이 많이 이루어져야만 문제를 해결할 수 있다는 것을 이해하게 되었습니다.
